How to Contribute

Quick steps to get started:

  1. Fork the repository on GitHub.
  2. Clone your fork: git clone https://github.com/your-username/hbtuHacktoberfest2025.git
  3. Pick issues labeled hacktoberfest or good-first-issue.
  4. Make changes, commit, and open a Pull Request.

Don't forget to follow the Code of Conduct and include clear commit messages.
